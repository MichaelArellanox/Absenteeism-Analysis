# Absenteeism Analysis 📈

### Objective
Respond to HR request of:
1. Provide a list of healthy individuals & low absenteeism for a bonus program
     - Company has a budget of $1000 USD to reward employees
2. Calculate wage increase or annual compensation for non-smokers
     - Company has a budget of $983,221 USD for all non-smokers
3. Create a dashboard to understand absenteeism, based upon an apporved wireframe

### Data set
- Absenteeism at Work [Absent Data](https://absentdata.com/data-analysis/where-to-find-data/)
